# PRISM Enterprise

**Current version:** V3.0 Foundation  
**Status:** Active development

PRISM Enterprise is a structured football match analysis and post-match learning framework. It combines quantitative models, tactical analysis, market interpretation, multi-model review, and systematic error classification.

## Core principles

1. Scientific honesty takes priority over attractive predictions.
2. Market odds are priors, not conclusions.
3. Match-result probability and handicap-cover probability must be separated.
4. All generated probabilities must pass mathematical consistency checks.
5. Tactical and contextual claims must be traceable to evidence.
6. Every completed match should produce a post-match review and an error classification.
7. GitHub is the single source of truth for confirmed PRISM rules, models, prompts, and reports.

## V3.0 architecture

- **Module A — Data Integrity**
- **Module B — Team Strength**
- **Module C — Attack & Defence**
- **Module D — Context & Motivation**
- **Module E — Dynamic Game State**
- **Module F — Market Interpretation**

## Standard workflow

1. Collect current match evidence.
2. Build the initial PRISM Match Data Sheet.
3. Run quantitative and tactical models.
4. Generate an independent Gemini research prompt.
5. Audit Gemini for factual, mathematical, tactical, and contextual consistency.
6. Produce the PRISM consensus forecast.
7. Conduct post-match scoring and error analysis.
8. Update weights or rules only after sufficient evidence.
